Youtrack

youTrack REST API Client

Installation

Add this line to your application's Gemfile:

gem 'youtrack'

And then execute:

$ bundle

Or install it yourself as:

$ gem install youtrack

Usage

client = Youtrack::Client.new do |c|
  c.url = "https://example.com"
  c.login = "root"
  c.password = "root"
end
